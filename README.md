# 🌸 Feliz Día de la Madre

Landing page sencilla y animada para el Día de la Madre, con fondo negro y pétalos cayendo.

## ¿Qué incluye?

- Hero a pantalla completa con título en tipografía serif
- Pétalos animados cayendo en CSS puro
- Círculos de luz difusa en el fondo
- Flor con animación de florecimiento al cargar
- Firma personalizada al pie

## Tecnologías

- HTML5 + CSS3 sin dependencias
- Google Fonts — Playfair Display + DM Sans
- CSS Keyframes para todas las animaciones

## Uso

Abre `index.html` en cualquier navegador. No requiere servidor ni instalación.

## Personalización rápida

| Qué | Dónde |
|---|---|
| Fecha | `.hero-eyebrow` en el HTML |
| Nombre / firma | `.footer-note` al final del `hero-content` |
| Color principal | `--rose-dark` en `:root` |
